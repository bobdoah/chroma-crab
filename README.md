# Chroma Crab
An [embassy-rs](https://embassy.dev/) project for a [Pimoroni Plasma 2350](https://shop.pimoroni.com/products/plasma-2350?variant=42092628246611) powered string light display.

# Usage
1. Hold boot and reset on the plasma board, whilst it's connected via USB.
1. `cargo run` to build and load the project using [`picotool`](https://github.com/raspberrypi/picotool).
